PHASER.IO

O Phaser é uma estrutura de jogo HTML5 de código aberto rápida, gratuita e divertida que oferece renderização WebGL e Canvas em navegadores da Web para desktop e dispositivos móveis. Os jogos podem ser compilados para iOS, Android e aplicativos nativos usando ferramentas 3rd party. Você pode usar JavaScript ou TypeScript para desenvolvimento.

Juntamente com a fantástica comunidade de código aberto, o Phaser é ativamente desenvolvido e mantido pelo Photon Storm. Como resultado de um suporte rápido e uma API amigável para desenvolvedores, o Phaser é atualmente um dos frameworks de jogos mais estrelados no GitHub.
